# TaskFlow 🚀

## About the Project
TaskFlow is a premium, high-performance task management application designed with a modern **Material 3 Midnight Glass** aesthetic. It provides a seamless experience for managing tasks, tracking progress, and handling complex task dependencies with real-time feedback and offline persistence.

## Tech Stack
- **Framework**: [Flutter](https://flutter.dev/) (3.27+)
- **State Management**: [Riverpod 2](https://riverpod.dev/)
- **Local Database**: [Hive](https://docs.hivedb.dev/)
- **Design System**: Material 3 with Custom Glassmorphism
- **Language**: Dart

## How to Clone
To get a local copy of this project, run the following command in your terminal:
```bash
git clone https://github.com/9mit/TASK_FLOW.git
```

## How to Run
Follow these steps to get the application running on your machine:

1. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

2. **Run the App on Mobile/Desktop/Default web browser**:
   ```bash
   flutter run
   ```

3. **Run on Specific Web Browsers**:
   - To run on **Google Chrome**:
     ```bash
     flutter run -d chrome
     ```
   - To run on **Microsoft Edge**:
     ```bash
     flutter run -d edge
     ```

4. **Run Tests** (Optional):
   ```bash
   flutter test
   ```
